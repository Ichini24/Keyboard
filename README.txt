Доброго времени суток, господа.
Это ПО написано для проверки клавиатурного почерка.
Сперва вам следует ввести свой придуманный пароль трижды в форму для того, чтобы система "запомнила" ваш клавиатурный почерк.
Пароль должен содержать:
	- ТОЛЬКО буквы латинского алфавита(не русского)
	- буквы нижнего регистра
	- буквы верхнего регистра
	- числа
	- спец. симолы
	- от 8 символов

После того, как вы верно ввели свой пароль трижды, можете вводить пароль в то же поле. После ввода пароля, в таблице появится строка, содержащая:
	1). Номер попытки ввода пароля
	2). Клавишный интервал - код введенного вами символа в ANSI таблице
	3). Эталон - эталонное время промежутка времени между нажатиями клавиш
	4). Измеренный - измеренное значение промежутка времени между нажатиями клавиш
	5). Делта - отклонение измеренного значения промежутка времени между нажатиями клавиш от этанола в процентах

Если вы хотите изменить текущий пароль, задать другие интервалы времени между нажатиями клавиш, нажмите на кнопку Reset.

Есть возможность изменить критерий системы по определению валидности интервала, выбрав его из выпадающего списка. Изначально установлен средний уровень сложности.
Если вы хотите сменить уровень сложности, требуется сначала нажать кнопку Reset, тогда все ваши данные будут утеряны и вы сможете сменить уровень сложности. 

Если вы ошиблись при вводе пароля, не используйте клавиши для удаления символов, т.к. система измеряет промежутки времени между нажатиями на клавиши и, если вы ввели какой-либо 
символ неправильно, а после удалили, эта попытка считается ложной, т.к. будет некорректно измерено время.
